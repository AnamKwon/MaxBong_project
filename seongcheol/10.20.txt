  목표 : 이미지 기반 AI 모델 학습을 위한 AI데이터 전/후처리 PJ

  배경 : 1. 다양한 이미지 기반 AI 학습 모델 개발 및 적용을 위해서는 딥러닝 모델을 위한 학습 데이셋이 매우 중요함.
          2. AI 모델의 성능은 학습 데이터의 저작 및 가공이 높은 영향을 미침
          3. 예를 들어 (중복데이터의 제거, 명확하지 않은 영상의 화질 개선, 동영상 데이터의 캡쳐, 데이터 병합, 부족한 학습 데이터의 량을 늘리기 위한
                          데이터 증강, 여러 학습 데이터 포맷의 변환 등이 필요함)
          4. 따라서 AI 모델 학습 및 개발에 필요한 여러가지 기능을 정의하고 하나의 프로그램으로 작성하여 다양한 이미지 기반 AI 모델 개발과 응용에 필요한
             종합적인 학습 데이터 가공 및 전/후 처리를 하나의 시스템에서 가능한 프로그램의 개발을 목표로 한다.

% 이미지 AI 모델을 개발하거나 응용을 하기위한 여러 도구를 통합 제공하는 시스템을 제작하는것

* 데이터셋 타입 변환 (레이블링 후 처리)

   MScoco (json), Pascal VOC, Yolo  <-> 학습 데이터포맷 변환 기능 있으면 좋겠다...
   
   1. COCO, PascalVOC → YOLO
      https://github.com/ssaru/convert2Yolo (깃허브 소스 링크, 다양한 포맷을 yolo타입으로 변환) https://deepbaksuvision.github.io/Modu_ObjectDetection/ (사용법 링크) 

   2. COCO → VOC
      https://gist.github.com/jinyu121/a222492405890ce912e95d8fb5367977 (coco_to_voc.py) 

   3. YOLO → VOC
      https://github.com/surserrr/yolo_to_coco (yolotovoc.py 파일)
      https://github.com/surserrr/yolo_to_coco (voctococo.py 파일) 

    4. YOLO → COCO
      바로 변환하는 것은 찾지 못했고 YOLO → PascalVOC → COCO 와 같이 VOC를 거쳐야함 
      https://github.com/surserrr/yolo_to_coco (yolo를 voc로 변환 후 coco로 변환하는 방법) 

   5. VOC → COCO
     https://github.com/shiyemin/voc2coco (voc2coco.py 파일) 
     https://github.com/surserrr/yolo_to_coco (voctococo.py 파일)
